## Overview

This is a Kaggle competition to predict whether a passenger will be transported to another dimension during the Spaceship Titanic's collision with the spacetime anomaly. The competition is hosted by Kaggle and the data is provided by thgit e Spaceship Titanic team.

## Data

The data is provided in the `data` directory. The data is split into a training set and a test set. The training set contains the features and the target variable. The test set contains the features only.
